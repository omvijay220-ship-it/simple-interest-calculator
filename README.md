# Simple Interest Calculator

This project is a simple interest calculator.

## Formula Used

Simple Interest = (Principal × Rate × Time) / 100

## Description

This calculator takes:
- Principal amount
- Rate of interest
- Time period

And calculates the simple interest using the formula.

## Example

If:
Principal = 1000  
Rate = 5%  
Time = 2 years  

Simple Interest = (1000 × 5 × 2) / 100  
Simple Interest = 100
